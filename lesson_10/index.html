Блок "Sticky-1"

Позиціонування в CSS — це спосіб контролю розташування елементів на веб-сторінці. Воно визначає, як елементи взаємодіють один з одним і з контейнером. Розуміння позиціонування є ключовим для створення адаптивних і гармонійних інтерфейсів. CSS надає п’ять основних типів позиціонування: static, relative, absolute, fixed і sticky. Кожен з них має свої особливості та сценарії використання. Static (за замовчуванням) Усі елементи за замовчуванням мають position: static. Це означає, що вони розташовуються у звичайному потоці документа — зверху вниз, згідно з порядком у HTML. Властивості top, right, bottom, left для статичних елементів не працюють. Цей тип позиціонування використовується майже для всіх елементів за замовчуванням. Relative (відносне позиціонування) Елемент з position: relative залишається в потоці документа, але його можна змістити від початкового місця за допомогою властивостей top, right, bottom, left. Наприклад, top: 20px зрушить елемент вниз на 20 пікселів від його звичайного положення. Relative-позиціонування часто використовується як контейнер для абсолютних елементів.

Блок "Sticky-2"

Absolute (абсолютне позиціонування) Елемент з position: absolute виводиться з потоку документа. Його положення визначається щодо найближчого предка з позиціонуванням відмінним від static (наприклад, relative, absolute або fixed). Якщо такого предка немає, елемент позиціонується відносно body. Absolute-позиціонування дозволяє точно розміщувати елементи, наприклад, спливаючі вікна або іконки. Fixed (фіксоване позиціонування) Елемент з position: fixed прикріплюється до вікна браузера, незалежно від прокрутки сторінки. Це корисно для хедерів, футерів або кнопок «Вгору». Fixed-елементи не впливають на розташування інших елементів і залишаються на місці при скролі. Sticky (липке позиціонування) Sticky-комбінація між relative і fixed. Елемент поводиться як relative, поки не досягне певної позиції під час прокрутки, після чого «прилипає» і веде себе як fixed. Це зручно для меню, що залишаються видимими при прокручуванні, або для закріплених заголовків таблиць. CSS позиціонування — це потужний інструмент для контролю розташування елементів на сторінці. Окрім базових типів позиціонування, існують тонкощі, які допомагають вирішувати складні завдання веб-дизайну. Одним із основних аспектів є комбінування позиціонування та flexbox або grid. Наприклад, елемент з position: absolute всередині flex-контейнера може поводитися інакше, ніж в звичайному контейнері. Важливо пам’ятати: абсолютне позиціонування завжди відноситься до найближчого предка з позиціонуванням. Якщо цього предка немає — елемент буде позиціонуватися відносно body.
